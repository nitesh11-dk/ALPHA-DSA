*(This is your usual custom ice-breaker, like the ones we did before)*

```

### **2A️⃣ Then: Book next step**

```

Batao kab sahi rahega?
```

```
Perfect! Time aur day batao — main link bhejta hoon.
```

### **2C️⃣ If they say NO → Offer help:**

```

```

```
