### **Question 3: Find the Transpose of a Matrix**  
**Problem:**  
Write a program to find the **transpose** of a given matrix. Transposing a matrix involves swapping its rows with columns.

**Example:**  
For a **2x3 matrix**:

```
Matrix:  
a11 a12 a13  
a21 a22 a23  
```

The **transposed matrix** will be:

```
a11 a21  
a12 a22  
a13 a23  
```
